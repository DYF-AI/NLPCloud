# NLPCloud

基于FastAPI的NLP云服务代码库(tensorflow 2.x)

- python 3.8
- tensorflow 2.3
- jieba

一、NLP基础算法

1. 词法分析

   - 分词
   - 词性标注
   - 实体识别
   - 拼写检查

2. 句法分析

   - 句法结构分析
   - 依存关系分析
   - 深层文法句法分析

3. 语义分析

   - 词语级别语义分析

     - 词义消歧
       - 基于词典的语义消歧
       - 有监督语义消歧
       - 无监督和半监督语义消歧

     - 语义表示和学习

   - 句子级别语义分析

      - 浅层语义分析
      - 深层语义分析

4. 文档分析

5. 词向量



二、NLP应用

1. 聊天机器人

2. 机器翻译

3. 信息抽取

4. 文本分类

5. 文本摘要

6. 知识图谱
